public __construct() : void

Esta función deserializa los datos del usuario almacenados en el superglobal $_SESSION, extrae y formatea la URL de la página actual, y verifica si el usuario tiene una base de datos asociada. Luego, accede a la configuración de la base de datos y del usuario, establece la conexión a la base de datos utilizando mysqli, y verifica si la conexión se realizó correctamente. Si hay errores durante el proceso, redirecciona a la página de inicio con un mensaje de error correspondiente. Además, muestra un mensaje de inicio de sesión si la variable de configuración indica que se deben mostrar las consultas SQL.

public crud(string $sql[, string $column = '' ][, string $id = '' ]) : bool

En caso de una operación de inserción, ejecuta la sentencia SQL y registra la operación en la auditoría. Muestra mensajes de inicio de sesión y errores en la consola según la configuración, y cierra la conexión a la base de datos después de cada operación. Si ocurren errores durante la ejecución, los registra en un archivo de registro y muestra mensajes de error en la consola.

public respaldar() : bool

Crea una ruta para guardar el archivo de respaldo y verifica si ya existe un respaldo para la fecha actual. Si no existe un respaldo para la fecha actual, ejecuta el comando de respaldo utilizando el comando mysqldump con los datos de usuario y la base de datos actual, guardando el archivo de respaldo en la ruta especificada. Retorna un valor booleano indicando si el respaldo se realizó con éxito o no.
