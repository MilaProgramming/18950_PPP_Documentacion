buscar()

class.Entidad.php : 48

Metodo que devuelve registros segun condicion tipo La función acepta tres parámetros opcionales: tipo, condiciones y orden.

public buscar([int $tipo = 0 ][, array<string|int, mixed> $condiciones = array() ][, array<string|int, mixed> $orden = array() ]) : mixed

El parámetro tipo determina si se busca un único registro (tipo = 1) o varios registros (tipo > 1). El parámetro condiciones es un arreglo que contiene las condiciones de búsqueda. El parámetro orden es un arreglo que especifica el orden en que se deben devolver los resultados. La función utiliza la clase Bdd para ejecutar consultas SQL y obtener los resultados. Primero, obtiene el nombre de la tabla y las columnas de la misma. Luego, construye una consulta SQL para buscar los registros de la tabla. Si se proporcionan condiciones de búsqueda, las agrega a la consulta SQL. También agrega un ordenamiento a la consulta si se especifica en el parámetro orden. Finalmente, ejecuta la consulta SQL y devuelve los resultados de acuerdo al tipo especificado. Retorna los resultados de la búsqueda si tiene éxito, de lo contrario retorna false.

Parameters
$tipo : int = 0

Tipo de consulta

  • 0 Por defecto
  • 1 Un unico registro
  • Mayor a 1 Listados de registros que cumplen condicion
$condiciones : array<string|int, mixed> = array()

Objeto de condicionales de consulta

$orden : array<string|int, mixed> = array()

Objeto de orden de consulta

Return values
mixed —

registros o registro

  • array Si el $tipo es -1 o >1
  • array Si el $tipo es 1, con el unico registro
  • false Si hay un error en la consulta

class.Entidad.php : 148

Metodo que devuelve registros segun condicion tipo La función acepta dos parámetros: sql y tipo.

public buscarSql(string $sql[, int $tipo = 0 ]) : mixed

El parámetro sql es la consulta SQL que se desea ejecutar. El parámetro tipo determina si se busca un único registro (tipo = 1) o varios registros (tipo != 1). La función utiliza la clase Bdd para ejecutar la consulta SQL y obtener los resultados. Si el tipo es 1, la función devuelve un único registro. Si el tipo es diferente de 1, la función devuelve una lista de registros. Retorna los resultados de la consulta SQL si tiene éxito, de lo contrario retorna false.
